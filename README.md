# AI职场行动教练

> 📝 本项目使用 Trae IDE 进行开发

## 项目简介

AI职场行动教练是一个Web应用，旨在为大三、大四及应届毕业生（尤其是非技术岗、跨专业求职学生）提供从岗位解读到简历面试的全链路AI化指导。

## 核心功能

### 1. 岗位专业匹配
- 展示具体岗位对应的专业
- 给出跨专业求职的难度评估
- 提供专业匹配度分析

### 2. 岗位翻译器
- 输入JD文本或链接
- 输出：真实工作场景 + 硬技能/软技能拆解 + 不适合什么样的人

### 3. 行动规划师
- 输入目标岗位 + 年级专业
- 输出：1-3个月分阶段行动清单 + 优先补的一项能力

### 4. 简历面试教练
- 粘贴简历片段 → AI诊断并改写
- 模拟面试问题 → 用户回答 → AI给出改进建议

## 安装与运行

### 前置要求
- Node.js 16.0+  
- npm 7.0+
- 硅基流动 API 密钥

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目仓库地址>
   cd CareerHelper
   ```

2. **安装前端依赖**
   ```bash
   cd client
   npm install
   cd ..
   ```

3. **安装后端依赖**
   ```bash
   cd server
   npm install
   cd ..
   ```

### 运行项目

1. **启动后端服务**
   ```bash
   cd server
   npm run dev
   ```
   后端服务将在 `http://localhost:3000` 运行

2. **启动前端服务**
   ```bash
   cd client
   npm run dev
   ```
   前端应用将在 `http://localhost:5173` 运行

## API配置

1. 打开应用后，点击导航栏中的「API配置」
2. 输入您的硅基流动 API 密钥
3. （可选）输入自定义 API 端点（默认为：https://api.siliconflow.cn/v1）
4. 点击「保存配置」按钮

## 使用指南

### 岗位专业匹配
1. 进入「岗位专业匹配」页面
2. 浏览不同岗位及其相关专业
3. 查看跨专业求职的难度评估
4. 点击岗位卡片查看详细信息

### 岗位翻译器
1. 进入「岗位翻译器」页面
2. 粘贴岗位描述或输入岗位链接
3. 点击「分析岗位」按钮
4. 查看分析结果，包括真实工作场景、硬技能/软技能拆解和不适合人群分析
5. （可选）点击「保存到本地」按钮保存分析结果

### 行动规划师
1. 进入「行动规划师」页面
2. 输入目标岗位、选择年级、输入专业
3. 点击「生成行动规划」按钮
4. 查看1-3个月的分阶段行动清单和优先补充的能力
5. （可选）点击「保存到本地」按钮保存行动规划

### 简历面试教练
1. 进入「简历面试教练」页面
2. **简历分析**：粘贴简历片段，点击「分析简历」按钮，查看诊断和改写建议
3. **面试模拟**：输入面试问题和你的回答，点击「获取改进建议」按钮，查看评价、改进建议和更好的回答示例
4. （可选）点击「保存到本地」按钮保存分析结果

## 注意事项

- 本项目使用硅基流动 API，需要有效的 API 密钥
- API 调用可能会产生费用，请合理使用
- 系统会对用户输入进行处理，但不会存储用户的个人信息
- 生成的内容仅供参考，建议结合自身实际情况进行调整
- 切换页面时，输入内容和生成结果会自动保存，刷新页面后也会保留
